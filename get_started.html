HOW TO DOWNLOAD PALLADIUM WALLET


1. Go to GitHub

Click one of the download wallet buttons to go to the GitHub page where you'll find the necessary links for downloading the wallet.


2. Download the Palladium Wallet on GitHub

Click on one of the links shown in the picture to download the Palladium Wallet for Windows or Linux.


3. Start Palladium

When the download is finished, start the software and accept the terms of service.
