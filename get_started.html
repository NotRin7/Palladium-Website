HOW TO DOWNLOAD PALLADIUM WALLET


1. Go on palladium-coin.com

Write in your browser https://palladium-coin.com to visit the palladium website.


2. Go to Github

Go thru clicking one of the download Wallet buttons to the Github paege where you find the links necessary for downloading the wallet.


3. Download the palladium wallet on github

click on one of the links showen in the picture to download the Palladium Wallet for windows or linux.


4. Start palladium

when the download is finished start the software and accept the terms of service.
